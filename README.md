# AI NOW EXCEL PROJECT

### Project Title: Electronics Sales Analysis

### Project Overview

### Tools Used
- Microsoft Excel for data cleaning, analysis, and visualization
- GitHub for portfolio building

### Data Cleaning and Preparations
The following data cleaning steps were carried out to ensure data quality and consistency:
-  Removed excess spaces and unnecessary symbols to standardize text entries
-  Standardized inconsistent text formats by converting values to proper case
-  Cleaned and converted date fields into appropriate date formats for analysis
-  Corrected inaccurate data entries by replacing them with the appropriate values

### Exploratory Data Analysis
###  Data Analysis
The following formula was used to derive total sales:
```
Total Sales = Units Sold × Unit Price
```
This was used to calculate revenue and support the analysis in the dashboard.
